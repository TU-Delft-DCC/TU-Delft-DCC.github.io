# Checklist for mentor

A new RSE could be assigned a DCC mentor to support the onboarding during the first months, especially if working together in a hands-on support project. The mentor will be a colleague from the DCC team, but not the direct supervisor. The mentor will be responsible for the following tasks:

::: {.callout-tip}
## Checklist
- [ ] Introduction to the DCC routines
- [ ] Tour of the buildings (IDE/Library, Faculty)
- [ ] How to reserve rooms (IDE/Library, Faculty)
- [ ] Plan meetings with relevant colleagues
  - [ ] ICT colleagues - from operations only relevant managers (datamanagement, security/privacy, Faculty IT managers, Innovation team)
  - [ ] Library colleagues - Research Data Management / RD Services 
  - [ ] 4TUResearch Data team (community manager/ technical team)
  - [ ] Data stewards
  - [ ] Open Science Community Delft Community manager (Tanya Yankelevich)
  - [ ] Library Training coordinator (Paula Martinez Lavanchy)
- [ ] Discuss and communicate working days (on campus/at home) and plan introduction meetings accoordingly 
- [ ] Go over meeting and events schedule
:::
